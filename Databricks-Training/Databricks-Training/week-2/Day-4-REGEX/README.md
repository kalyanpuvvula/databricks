📘 SQL Regex Practice Assignment

This assignment focuses on implementing **Regular Expressions (Regex)** in SQL for pattern matching, extraction, and text analysis using different types of structured and mixed data.

🚀 Topics Covered
- Regular Expressions in SQL
- Pattern Matching
- String Extraction
- Email Parsing
- Phone Number Analysis
- Alphanumeric Data Processing

🛠 Regex Functions Used
- `REGEXP_SUBSTR()`

📚 Queries Included
- Extract numbers from strings
- Extract alphabets from mixed values
- Find prefixes and suffixes
- Extract email usernames and domains
- Extract country codes from phone numbers
- Extract continuous numeric patterns
- Extract characters from beginning and end of strings

🗂 Database Table
- `regex_practice`

📌 Columns Used
- `id`
- `full_text`
- `email`
- `phone`
- `mixed_value`

🎯 Learning Outcome
- Understand Regex implementation in SQL
- Perform advanced string extraction
- Work with real-world text patterns
- Improve SQL pattern matching skills
- Practice data parsing and validation

💻 Technologies Used
- SQL
- Oracle SQL / MySQL 8+
