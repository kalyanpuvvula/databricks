# Business Pipeline & Analytics using PySpark

## Project Overview

This project is developed as part of the **Phase 4 – Mini Project: Business Pipeline & Analytics**. The objective is to build an end-to-end data pipeline using **PySpark** and **Spark SQL** to process customer and sales data and generate meaningful business insights.

---

## Objectives

- Load customer and sales datasets.
- Perform data cleaning.
- Process business data using PySpark.
- Analyze sales and customer information.
- Generate reports for business decision-making.
- Save the final output.

---

## Technologies Used

- Python 3.8
- PySpark 3.5.5
- Spark SQL
- CSV Files
- Apache Spark

---

## Datasets Used

- customers.csv
- sales.csv

---

## Data Cleaning

The following data cleaning steps were performed:

- Removed duplicate records.
- Removed rows with null customer IDs.
- Converted `total_amount` to Double data type.
- Removed records with negative sales amounts.

---

## Tasks Completed

### Task 1 – Daily Sales
- Calculated total sales for each day.

### Task 2 – City-wise Revenue
- Calculated total revenue generated by each city.

### Task 3 – Top 5 Customers
- Identified the top five customers based on total spending.

### Task 4 – Repeat Customers
- Found customers who placed more than one order.

### Task 5 – Customer Segmentation
Customers were classified into:

- Gold (Total Spend > 10000)
- Silver (5000 – 10000)
- Bronze (< 5000)

### Task 6 – Final Reporting Table
Generated a final report containing:

- Customer Name
- City
- Total Spend
- Order Count
- Customer Segment

### Task 7 – Save Output
Saved the final report as a CSV file.
