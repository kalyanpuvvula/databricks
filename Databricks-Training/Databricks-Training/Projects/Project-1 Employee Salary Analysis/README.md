# 💰 Project 1: Employee Salary Analytics

## 📌 Overview

This PySpark project analyzes employee salary data to generate meaningful business insights using DataFrame transformations, aggregations, and window functions.

## 🛠️ Tech Stack

* 🐍 Python
* ⚡ PySpark
* 🧱 Databricks

## ✨ Features

* 🏆 Find Top 3 Highest Salaries Department-Wise
* 🌍 Calculate Average Salary City-Wise
* 📊 Categorize Employees into Salary Bands
* 📅 Generate Yearly Salary Reports
* 🎯 Identify Employees Earning Above Department Average

## 🔄 Workflow

```text
Employee Data
      ↓
Salary Analysis
      ↓
Department Insights
      ↓
City-Wise Reporting
      ↓
Yearly Trends
      ↓
Business Insights
```

## 📈 Output

* ✅ Top Earners by Department
* ✅ City-Wise Average Salary Report
* ✅ Salary Band Classification
* ✅ Yearly Salary Summary
* ✅ Above-Average Salary Analysis
